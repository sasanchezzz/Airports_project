# REST API приложение для базы данных Авиаперевозки

Микросервис для управления данными авиакомпании, предоставляющий REST API для работы с билетами, рейсами, самолетами и аэропортами. Разработан с использованием современного стека технологий с акцентом на чистоту кода и обработку ошибок.

## Стек
**`Для микросервиса:`** Python 3.12.3, Asyncpg, FastAPI, fastapi-pagination, SQLAlchemy 2.0, Pydantic, PostgreSQL, Uvicorn, Git

**`Для тестов:`** Pytest, Pytest-asyncio, Testcontainers[postgres]

## Функциональность
**`Для пассажиров:`**
- **Создание билета**
- **Удаление существующего билета**
- **Получение информации о посадочном талоне по номеру билета**

**`Для авиакомпании:`**
- **Получение аналитики за определенный промежуток времени**
- **Внесение информации о новом самолете**
- **Обновление данных о самолете по его коду**
- **Удаление данных о самолетах**
- **Внесение информации о новом самолете**
- **Обновление данных о билете пассажира**

| ├── **`app/`** | Папка с кодом приложения |
| │   ├── **`api/`** | Папка с API логикой |

| │   │    ├── **`v1/`** | Версионирование API (версия 1) |

# Установка
1. Клонируйте репозиторий
```bash
git clone https://github.com/sasanchezzz/Airports_project.git
cd Airports_project
```
2. Загрузите приложение через docker-compose
```bash
docker-compose up --build
```
3. Запустите приложение с помощью файла start.py
```bash
python run start.py
```
4. Проверьте работу API
```
Откройте в браузере: http://localhost:8000/docs
```
